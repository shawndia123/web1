팁 사이트 & 단축키

홈짱닷컴 윈도우 단축키
  1. 커서가 위치한 같은 단어 전체 선택: Ctrl +F2
  2. 주석: Ctrl + /
  3. 커서가 위치한 같은 단어 반복 선택: Ctrl + D
  4. 가상 데스크톱: Ctrl + 윈도우 + D(생성), 방향키(전환), F4(삭제)
  5. 바탕화면: 윈도우 + M
  6. 웹 분할: 윈도우 + 방향키
  7. 로그아웃: Ctrl + L
  8. 줄 복사: 커서 위치에서 (선택된 것이 없는) Ctrl + C
  9. 줄 잘라내기: 커서 위치에서 (선택된 것이 없는) Ctrl + X
  10. 줄 복제: Shift + Alt + Down (아래에), Shift +Alt + Up (위에)
  11. 줄 이동: Alt + Down(아래로), Alt+Up(위로)
  12. 줄 삭제: Ctrl + Shift + K
  13. 커서 위에 빈줄 생성: Ctrl + Enter
  14. 커서 아래에 빈줄 생성: Ctrl + Shift + Enter
  15. 커서가 위치한 라인 선택: Ctrl + I
  16. 라인 들여쓰기: Ctrl +]
  17. 라인 내여쓰기: Ctrl +[
  18. 선택 영역 주석: Shift + Alt + A
  19. 선택영역 코드 정렬: Ctrl + K + F
